## 🎮 サーバーの特徴

<CardGrid stagger>
  <Card title="テレポート機能" icon="rocket">
    `/tpa` コマンドなどで、プレイヤー同士の移動がスムーズに行えます。
    [詳細を見る](/features/tp/)
  </Card>
  <Card title="土地保護" icon="shield">
    金のシャベルを使って自分の土地を保護し、荒らしから守ります。
    [詳細を見る](/features/claims/)
  </Card>
  <Card title="チェスト・ドア保護" icon="framer-lock">
    看板を貼るだけで、チェストやかまど、ドアに自分専用の鍵をかけられます。
    [詳細を見る](/features/cdhogo/)
  </Card>
  <Card title="特殊スキル" icon="star">
    採掘や戦闘を繰り返すことでレベルアップ！自分だけのステータスを強化しよう。
    [詳細を見る](/features/skills/)
  </Card>
  <Card title="一括破壊機能" icon="pencils">
    木こりや鉱石掘りもサクサク。作業のストレスを減らして、建築や冒険に集中できます。
    [詳細を見る](#)
  </Card>
  <Card title="エッグキャッチャー" icon="starlight">
    卵を投げつけることでモブを「スポーンエッグ」として捕獲できます！
    [詳細を見る](/features/eggc/)
  </Card>
</CardGrid>
