MiLight Serverへ参加するための設定情報です。
現在は**開発中（アルファ版）**として運用されています。

---

## ☕ Java版 (PC)
PC（Windows / Mac / Linux）でプレイされている方はこちらです。

### 接続情報
* **推奨バージョン**: `1.21.1` , `1.20.x`
* **対応バージョン**: `1.8.x` ～ `1.21.x` (最新)
* **サーバーアドレス**: `java.mlms.n1e.net`

:::tip[バージョンのヒント]
サーバーはマルチバージョンに対応していますが、最新のブロックや機能を正しく表示するために、可能な限り **1.21.1** 以上での参加を推奨します。
:::

---

## 📱 BE / 統合版 (スマホ・コンソール)
スマホ（iOS/Android）、タブレット、Switch、PS4/5、Xboxなどでプレイされている方はこちらです。

参加方法は「サーバー追加」と「フレンド申請」の2通りあります。

### 1. サーバー追加で参加
「サーバー」タブから「サーバーを追加」を選択し、以下の情報を入力してください。

| 項目 | 設定内容 |
| :--- | :--- |
| **サーバー名** | MiLight Server (任意) |
| **アドレス** | `mlms.n1e.net` |
| **ポート** | `12751` |

:::caution[ポート番号に注意]
標準の「19132」のままでは接続できません。必ず **12751** に書き換えてください。
:::

### 2. フレンド申請で参加
SwitchやPS4/5など、直接サーバーアドレスを入力できない場合は、以下の公式アカウントにフレンド申請を送ることで「フレンド」タブから参加できます。

* **ゲーマータグ**: `mlms2`

---

## 🛠️ 接続できないときは？

もしサーバーに入れない場合は、以下の点を確認してください。

1.  **アドレス・ポートの入力ミス**: `mlms.n1e.net` やポート `12751` が半角英数字で正しく入力されているか。
2.  **Microsoftアカウント**: BE版の場合、サインインしている必要があります。
3.  **インターネット接続**: Wi-Fiやモバイルデータ通信が安定しているか。
4.  **メンテナンス**: 公式Discordなどでメンテナンス情報が出ていないか確認してください。
