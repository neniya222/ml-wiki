MiLight Serverに導入されている **AuraSkills** では、生活や戦闘のあらゆる行動が「スキル」として蓄積されます。
レベルが上がるごとにステータスが強化され、サバイバルがより有利になります！

---

## 📜 スキル一覧
それぞれの行動を行うことで経験値（XP）が貯まり、レベルアップします。

### 🌿 生活系
| スキル名 | レベルアップ条件 |
| :--- | :--- |
| **Farming**（農耕） | 作物を収穫する |
| **Foraging**（伐採） | 木を斧で切る |
| **Mining**（採掘） | 岩石や鉱石をツルハシで掘る |
| **Fishing**（釣り） | 釣り竿で魚を釣る |
| **Excavation**（掘削） | 土や砂をシャベルで掘る |

### ⚔️ 戦闘・冒険系
| スキル名 | レベルアップ条件 |
| :--- | :--- |
| **Fighting**（戦闘） | 剣などの近接武器で敵を倒す |
| **Archery**（射撃） | 弓やクロスボウで敵を射抜く |
| **Defense**（防御） | モンスター等からダメージを受ける |
| **Agility**（俊敏性） | 移動中に落下ダメージを受ける（パルクール等） |

### ⚗️ 特殊系
| スキル名 | レベルアップ条件 |
| :--- | :--- |
| **Enchanting**（付与） | アイテムにエンチャントを施す |
| **Alchemy**（錬金術） | ポーションを調合する、または使用する |

## 💻 スキル関連のコマンド
チャット欄に入力して実行してください。

### 基本操作
* **`/skills`** : スキルメニューを開きます（進行状況の確認）。
* **`/stats`** : 自分の現在のステータス詳細を表示します。
* **`/[スキル名]`** : 特定スキルの詳細を直接開きます（例: `/farming`）。
* **`/mana`** : 現在のマナ残量を確認します。
* **`/sk claimitems`** : インベントリがいっぱいで受け取れなかった報酬を受け取ります。

### 表示・設定
* **`/sk lang [言語コード]`** : 表示言語を自分専用に変更します。
* **`/sk toggle`** (または `/abtoggle`) : 画面下の表示（アクションバー）のON/OFFを切り替えます。
  :::note
  初期状態では **OFF** になっています。レベルアップを画面で確認したい場合はONにしましょう！
  :::

### ランキング
* **`/sk top`** : 全プレイヤーの合計レベルランキングを表示します。
* **`/sk top [スキル名]`** : 特定スキルのランキングを表示します。
* **`/sk rank`** : 自分の現在の順位を確認します。

---

:::tip[経験値の稼ぎ方を知りたい時は]
**`/sk sources [スキル名]`** と入力すると、そのスキルの経験値を効率よく稼ぐ方法の一覧がゲーム内で表示されます！
:::
