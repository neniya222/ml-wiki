**MyPet** は、マインクラフトの世界にいるモブを自分の「ペット」として飼育できる非常に強力なシステムです。
ペットはレベルアップすることで、荷物持ちや戦闘の援護、さらには乗り物として活躍してくれます。

---

## 🐾 1. ペットを捕まえる (Taming)

MyPetでは、野生のモブを直接捕まえることで自分のペットにします。

* **捕獲方法**: 捕まえたいモブの体力を削り（推奨）、手に**リード（首輪）**を持って左クリックで叩きます。
* **注意点**: 体力が高いままだと捕獲に失敗することがあります。死なない程度に弱らせてから叩くのがコツです。

---

## 🌟 2. 育成とスキルツリー (Skilltrees)

ペットは経験値（XP）を得ることでレベルアップし、特殊な能力を習得します。
`/pcst` コマンドで、ペットにどのような役割を持たせるか選択しましょう。

| スキルツリー | 特徴 |
| :--- | :--- |
| **Combat / Fighter** | 戦闘特化。攻撃力が大幅に上昇します。 |
| **Utility** | 探索特化。専用インベントリ（バックパック）が使えます。 |
| **Ride / Mount** | 移動特化。ペットの背中に乗って移動できます。 |
| **Farm** | 農業補助。作物の収穫などを手伝ってくれます。 |
| **PvP / Tank / Mage** | 対人戦や防御、魔法攻撃などに特化した特殊な職種です。 |

:::tip[レベル上げのコツ]
ペットを連れた状態でモンスターを倒すと、ペットに経験値が入ります。
:::

## 📜 3. 主要コマンド一覧

ペットの管理に使う便利なコマンドです。

| 長いコマンド | 短縮形 | 内容 |
| :--- | :--- | :--- |
| `/petinfo` | `/pinfo` | ステータス確認（体力、経験値、空腹度） |
| `/petname [名前]` | なし | ペットに名前を付ける（カラーコード対応） |
| `/petcall` | `/pc` | 離れた場所にいるペットを呼び出す |
| `/petsendaway` | `/psa` | ペットを一時的に引っ込める |
| `/petbehavior` | `/pb` | 行動モード変更（攻撃的、中立、逃走など） |
| `/petinventory` | `/pinv` | インベントリを開く（Utilityスキルが必要） |
| `/petrelease` | なし | ペットを野生に返す（※二度と戻りません） |

---

## 🏠 4. 複数ペットの管理

MiLight Serverでは、複数のペットを使い分けることが可能です。

* **`/petstore`**: 現在連れているペットを「ペット倉庫」に預けます。
* **`/petswitch`**: 預けているペットのリストを表示し、別のペットに切り替えます。

---

## 🍎 5. ケアとリスポーン

* **エサ**: 右クリックでそのモブの好物（肉や野菜など）を与えると、空腹度と体力が回復します。
* **死亡**: 体力が0になるとペットは死んでしまいますが、一定時間後に自動で復活（リスポーン）します。デスペナルティはありませんのでご安心ください。

---

## 💡 どっちのペットがおすすめ？
* **序盤**: 足の速い「馬」を **Ride** スキルにして移動手段にするのがおすすめ！
* **中盤**: 攻撃力の高い「狼」や「アイアンゴーレム」を **Combat** にして狩りの相棒にしましょう。
