## 🛡️ 土地保護の方法

金のシャベルを使用して、対角線上の2点を指定することで範囲を保護できます。

### 手順
1. **[起点]** 保護したい範囲の角にあるブロックに視線を合わせます。
   <Image src={img1} alt="起点を選択" />
2. 金のシャベルを持って**右クリック**します。
   <Image src={img2} alt="シャベルで右クリック" />
3. **[終点]** 対角線上にある反対側の角のブロックに視線を合わせます。
   <Image src={img3} alt="終点を選択" />
4. 再度、金のシャベルで**右クリック**します。
   <Image src={img4} alt="保護完了" />

## 📐 保護範囲の変更
「家を広げたい」「少し範囲をずらしたい」ときは、以下の手順で微調整できます。

1. **範囲の可視化**: 棒（スティック）を持って、保護エリア内を**右クリック**します。地面にグロウストーン等で枠が表示されます。
2. **位置の選択**: 金のシャベルを持ち、動かしたい角にある**グロウストーンを右クリック**します。
3. **移動先を指定**: 新しく角にしたい場所のブロックを**右クリック**すると、範囲が更新されます。

---

## 🗑️ 土地保護の削除
保護が不要になった場合は、以下の手順で削除できます。

1. 削除したい保護地の中に立ちます。
2. チャット欄で `/abandonclaim` と入力して実行します。

:::danger[注意]
保護を削除すると、誰でもブロックを壊したりチェストを開けたりできる状態になります。
:::

## 📜 コマンド一覧
### 権限・管理コマンド
| コマンド | 短縮形 | 内容・説明 |
| :--- | :--- | :--- |
| `/AbandonClaim` | なし | 現在立っている場所の保護を削除します。 |
| `/AbandonAllClaims` | なし | **自分が所有する全ての土地保護を削除**します。 |
| `/ClaimExplosions` | なし | 保護地内での爆発（TNT/クリーパー）を許可/禁止します。 |
| `/TrustList` | なし | 現在の土地で許可を与えているプレイヤー一覧を表示します。 |
| `/ClaimsList` | なし | 自分が所有している保護地の一覧を表示します。 |

### プレイヤーへの権限譲渡（Trust）
| コマンド | 短縮形 | 与えられる許可レベル |
| :--- | :--- | :--- |
| `/Trust [名]` | `/t` | **フル権限**: 建築・破壊・チェスト・レバー等すべて。 |
| `/ContainerTrust [名]` | `/ct` | **コンテナ権限**: チェスト・作業台・動物・ボタン等。 |
| `/AccessTrust [名]` | `/at` | **アクセス権限**: ボタン・レバー・ベッドの使用のみ。 |
| `/PermissionTrust [名]` | `/pt` | 自分が持つ許可レベルを他人に分け与える権限。 |
| `/UnTrust [名]` | `/ut` | 指定したプレイヤーの全権限を取り消します。 |
| `/UnTrust All` | なし | **その土地にいる全員**の権限を一括で削除します。 |

### 高度な設定・その他
| コマンド | 短縮形 | 内容・説明 |
| :--- | :--- | :--- |
| `/SubdivideClaims` | `/sc` | 土地を細分化（サブクレーム）モードに切り替えます。 |
| `/BasicClaims` | `/bc` | 通常の土地保護モードに戻します。 |
| `/RestrictSubclaim` | `/rsc` | 親の土地の権限を継承しないよう制限します。 |
| `/BuyClaimBlocks` | なし | お金で保護できる面積（ブロック数）を購入します。 |
| `/SellClaimBlocks` | なし | 余っている保護面積を売却してお金にします。 |
| `/GivePet` | なし | 飼いならした動物を他人に譲渡、または手放します。 |
| `/Trapped` | なし | 他人の土地に閉じ込められた際に脱出します。 |
| `/UnlockDrops` | なし | 自分が死んだ際のドロップ品を誰でも拾えるようにします。 |

### プレイヤーの非表示・無視
| コマンド | 内容 |
| :--- | :--- |
| `/Ignore [名]` | 指定したプレイヤーのチャットを非表示にします。 |
| `/UnIgnore [名]` | 無視を解除します。 |
| `/IgnoreList` | 現在無視しているプレイヤーの一覧を表示します。 |
