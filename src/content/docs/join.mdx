MiLight Serverへ参加するための設定情報です。
現在は**アルファ版**として運用されています。

---

## ❓ 参加方法 (アルファ版)

お使いのデバイスに合わせて、以下の情報を入力して追加してください。

### ☕ Java版
PC（Windows/Mac/Linux）でプレイされている方はこちらです。

| 項目 | 設定内容 |
| :--- | :--- |
| **サーバー名** | MiLight Server (任意) |
| **サーバーアドレス** | `mlms.n1e.net` |

---

### 📱 BE (統合)版
スマホ（iOS/Android）、タブレット、Switch、PS4/5、Xboxなどでプレイされている方はこちらです。

| 項目 | 設定内容 |
| :--- | :--- |
| **サーバー名** | MiLight Server (任意) |
| **サーバーアドレス** | `mlms.n1e.net` |
| **ポート** | `25575` |

:::caution[注意]
BE版から参加する場合、**ポート番号（25575）**を正しく入力しないと接続できません。標準の「19132」から書き換えてください。
:::

---

## 🛠️ 接続できないときは？

もしサーバーに入れない場合は、以下の点を確認してください。

1. **アドレスの入力ミス**: `mlms.n1e.net` の綴りが合っているか確認してください。
2. **バージョンの確認**: サーバーが対応しているマイクラのバージョンと一致しているか確認してください。
3. **ネット環境**: Wi-Fiやモバイルデータ通信が安定しているか確認してください。
4. **サーバーの起動状態**:サーバーがメンテナンス中、停止中ではないか確認してください。
