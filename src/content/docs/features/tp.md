サーバー内での移動を便利にするコマンドを紹介します。
使い方は簡単で、チャット欄にコマンドを入力して送信するだけです。

---

## 🏠 ホーム機能（/home系）
特定の場所（家など）を記録して、いつでもそこに戻れる機能です。

### 基本的な使い方
| コマンド | 内容 |
| :--- | :--- |
| `/home` | 最後に寝たリスポーン地点（ベッド）に戻ります。 |
| `/home [名前]` | 設定した「ホームID」の場所にテレポートします。 |
| `/sethome [名前]` | 現在地を「ホームID」として保存します。 |
| `/delhome [名前]` | 保存したホームを削除します。 |
| `/homes` | 現在設定されているホームの一覧を表示します。 |

:::tip[ホーム設定のコツ]
`/sethome house` や `/sethome shop` のように、名前に英数字を使って場所を区別すると便利です！
:::

---

## 🌍 スポーン地点（/spawn）
サーバーの初期地点（みんなが最初に降り立つ場所）に戻るコマンドです。

### 基本的な使い方
* **`/spawn`**
    * 初期スポーン地点にテレポートします。迷子になった時や、共有施設に行きたい時に使いましょう。

---

## 🤝 プレイヤー間移動（/tpa系）
他のプレイヤーの場所へ行ったり、自分の場所に呼んだりする機能です。相手が承認（許可）することでテレポートが完了します。

### 申請を送る
| コマンド | 内容 |
| :--- | :--- |
| `/tpa [MCID]` | 指定したプレイヤーの場所へ行く申請を送ります。 |
| `/tpahere [MCID]` | 指定したプレイヤーを自分の場所に呼ぶ申請を送ります。 |
| `/tpcancel` | 送信した申請をキャンセルします。 |

### 申請を受ける・管理する
| コマンド | 内容 |
| :--- | :--- |
| `/tpayes` | 届いているテレポート申請を**承認（許可）**します。 |
| `/tpano` | 届いているテレポート申請を**拒否**します。 |
| `/tpalist` | 現在届いている保留中のリクエストを一覧表示します。 |

:::caution[注意点]
* 相手が忙しい時や、迷惑になりそうな時は `/tpa` を控えるなど、マナーを守って利用しましょう。
:::
